# Dashboard Penjualan Online Retail

Dashboard Business Intelligence (BI) untuk menganalisis data transaksi perdagangan elektronik menggunakan Streamlit dan Plotly.

## Instalasi

1. Clone repository

2. Install dependencies yang diperlukan:

```bash
pip install -r requirements.txt
```

3. Jalankan aplikasi dengan perintah:

```bash
streamlit run app.py
```
